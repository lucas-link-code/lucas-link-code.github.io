About Me

I lead malware defense operations at a major global financial institution, serving as Vice President of Information Security and EMEA regional lead. My daily work involves reverse engineering advanced malware, building detection systems at scale, and pioneering the use of GenAI tools to transform how security teams operate.

I specialize in the full spectrum of malware analysis across Windows and Linux platforms. I've built and deployed over 120 YARA rules and 120 EKFiddle detection rules to production systems, processed more than 16,000 indicators of compromise proactively, and led the response to emerging threats including ClickFix, ClearFake, and supply chain attacks targeting package repositories.

What drives me is the conviction that AI will fundamentally reshape cybersecurity. I've built MCP-integrated tools connecting large language models to traffic analysis platforms, developed AI Copilot proof-of-concepts for threat hunting, and I work to prepare teams for an AI-augmented future. Based in Wales, working from Chester, I bring the same intensity to mentoring analysts and sharing knowledge as I do to dissecting binaries.

Core Expertise

Malware Reverse Engineering

Deep binary analysis across PE, ELF, Delphi, .NET, and VBA formats. Static analysis with PE headers and metadata tools. Dynamic analysis in isolated lab environments with API hooking and process monitoring. Full reverse engineering using IDA Pro, Ghidra, and x32/x64dbg.

Detection Engineering

Creator of 120+ EKFiddle regex rules and 120+ YARA rules deployed to NSX Defender production sandboxes. Automated conversion pipelines from Suricata and Snort to EKFiddle and YARA formats. Domain reputation extensions for off-network analysis environments used by the global team.

GenAI Security Innovation

Built an MCP server integrating Fiddler traffic capture with Gemini LLM for natural language malware traffic analysis. Developed 5 GenAI agent use cases for security operations. Created AI Copilot systems for CrowdStrike FQL, LogScale CQL, and Splunk queries. Active in Shadow AI detection research.

Proactive Threat Hunting

Led operations targeting ClickFix, ClearFake, and Fake Update campaigns. Configured 40+ URLScan proactive alerts and 30+ VirusTotal YARA hunt rules. Processed 16,000+ IOCs proactively, mitigating 2,500+ threats before impact. Built C2 extraction tooling for large-scale intelligence processing.
